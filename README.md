# Практическрая работа №3
### Вариант №15: зоопарк

## Инструкция по сборке
1. Скачать и распаковать проект
2. Открыть в терминале папку с проектом
3. Собрать проект:
   ```
   mvn compile
   ```
4. Запустить проект:
   ```
   mvn exec:java -q
   ```

_Для сборки необходимо иметь [Maven](https://maven.apache.org/download.cgi) на компьютере_
